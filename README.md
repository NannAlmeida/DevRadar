# DevRadar

**Linguagens de programação utilizadas:**
- Javascript

**Bibliotecas utilizadas:**
- React Native
- React JS

**Ferramentas utilizadas:**
- NodeJS
- Yarn
- GitHub API
- Expo
- Visual Studio Code

**Descrição de execução do código:** 

- É necessário ter o NodeJS versão 12.13.0 LTS.
- É necessário a instalação das bibliotecas no Node: axios, cors, express, mongoose, socket.io
- É necessário a instalação das bibliotecas no ReactJS: Nenhuma
- É necessário a instalação das bibliotecas no React Native: axios, expo-location, react-native-gesture-handler, react-native-maps, react-native-reanimated, react-native-safe-area-context, react-native-screens, react-native-web, react-native-webview, react-navigation, react-navigation-stack, socket.io-client.
- Acesso a internet.
 

- Configurar a porta do servidor do node no arquivo src/index.js.
- Digitar no terminal "yarn dev" (sem aspas) para executar o BackEnd (NodeJS).
- Digitar em outra janela do terminal "yarn start" (sem aspas) para executar o ReactJS.
- Digitar em outra janela do terminal "yarn start" (sem aspas) para executar o React Native.
- Acessar no navegador localhost:porta configurada.


- Acessar o site.
- Aceitar que ele pegue sua localização.
- Colocar suas informações e clicar em Salvar.
- Executa o app dentro do Expo (Necessário app do Expo instalado no dispositivo móvel).
- Na barra digita a tecnologia que procura e aperta no botão ao lado direito.
- Aperta na foto do dev e irá aparecer as informações, apertando no balão das informações a aplicação é direcionada ao GitHub do Dev.
